Sample Email Details
Sender: support@paypa1-security.com
Subject: Urgent! Account Suspended
Link: http://paypal-verification-alert.ru/login
 Phishing Indicators Found
Spoofed Email Address
The sender uses "paypa1" instead of "paypal" (number 1 replacing letter l), indicating email spoofing and typosquatting.
Urgent/Threatening Language
The subject line creates panic by saying the account will be suspended.
Suspicious External Link
The link uses a ".ru" domain which is not associated with the official PayPal website.
Mismatched URL
The displayed link appears related to PayPal, but it redirects to a different domain.
Social Engineering Tactic
The email uses fear-based manipulation to trick users into clicking the malicious link.
